=== SLIDE 11 — PITCLIPSE / PIT: MUTACIÓN ===
Header badge: "ANÁLISIS 4" en #DA3633 | Título: "🔴 Pitclipse (PIT) — Pruebas de Mutación"
Layout: arriba → concepto | centro → tabla de resultados | abajo → caso de estudio

Concepto visual (3 pasos con flechas):
  [Tu código] → [PIT introduce mutante] → [¿Las pruebas fallan?]
  Si SÍ → "Killed ✅ — Prueba eficaz" (verde)
  Si NO → "Survived ❌ — Prueba débil" (rojo)

Cómo ejecutar:
  Clic derecho sobre proyecto → PIT → PIT Mutation Test
  Nota: "Puede tardar varios minutos. El reporte se abre en el navegador interno."

Tabla de resultados (3 filas):
  Killed | 🟢 Verde | Mutante detectado — prueba eficaz
  Survived | 🔴 Rojo | Mutante no detectado — prueba débil
  No coverage | ⚪ Gris | Línea sin ninguna prueba asociada

Caso de estudio — mutante superviviente:
  PIT cambió: monto > 0  →  monto >= 0 (en consignar)
  Ninguna prueba lo detectó → Survived ❌
  Solución:
    @Test
    public void shouldNotConsignarWhenMontoCero() {
        CuentaBancaria cuenta = new CuentaBancaria();
        cuenta.consignar(0.0);
        assertEquals(0.0, cuenta.getSaldo(), 0.01);
    }

Bloque "ACTIVIDAD" rojo: "⏸️ Cuando veas un mutante Survived, pausa y escribe la prueba que lo mataría."
